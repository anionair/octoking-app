# 문어왕 앱 업로드 파일

이 폴더 안의 파일들을 GitHub 저장소에 그대로 올리면 됩니다.

GitHub Pages에서 주소가 생기면 핸드폰에서 그 주소를 열어 테스트할 수 있습니다.

## 이번 버전 변경

- 문어왕 답변은 위쪽 말풍선으로 표시됩니다.
- 사용자가 입력한 질문은 문어왕 아래 파란 말풍선으로 표시됩니다.
- 마이크와 입력창을 문어왕 바로 아래로 옮겼습니다.
- `목소리 켜기` 버튼을 먼저 누르면 문어왕 목소리를 들을 수 있습니다.
- 선장/궁금/용감/다정 선택을 삭제했습니다.
- 나이와 답변 길이 선택을 삭제하고 8~10세/보통으로 고정했습니다.
- 문어왕을 터치하면 깜짝 놀라기, 먹물, 피하기 반응이 나옵니다.
